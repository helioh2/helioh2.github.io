Exercise: Completing Partial Code from Circles of Evaluation

Directions: Each Circle of Evaluation on the left has been partially-converted to code on the right. Finish the code by filling in the gaps, so that it matches the Circle.
  • (+ 16(* 6-3))
    (+ (* 6))
  • (- (+ 2513)(* 24))
    (- (+ 13)(4))
  • (* (+ 104)28)
    ((+ 4))
  • (* 13(/ 7(+ 2-4)))
    (* (/ (+ 2-4)))
  • (+ (/ (+ 81)3)(- 53))
    (+ (- 3))
